Write a JavaScript program to check whether a string is a palindrome or not. function isPalindrome(str) { const cleanStr = str.toLowerCase().replace(/[^a-z0-9]/g, ""); return cleanStr === cleanStr.split("").reverse().join(""); } // Example usage: console.log("4:", isPalindrome("Racecar")); // Output: true // 5. Write a JavaScript program to count the number of vowels in a given string. function countVowels(str) { return (str.match(/[aeiouAEIOU]/g) || []).length; } // Example usage: console.log("5:", countVowels("Space Exploration")); // Output: 7
